# 更新日志

本项目从 `1.0.0` 开始记录版本变更。每次发布前都要在这里新增对应版本的条目，并同步更新 README 中列出的版本号位置。

## 1.0.1 - 2026-06-13

- 修复鸿蒙手机/PWA 中点击“进入横屏”后可能卡在竖屏提示层的问题。
- 横屏按钮现在会优先请求系统全屏和原生横屏锁定。
- 当浏览器拒绝原生横屏锁定时，启用软件横屏模式，把练习界面旋转成可用的横屏画布。
- 增加 PWA 新版本提示，发现新版本时可点击“立即更新”切换到新版缓存。
- 将 Service Worker 离线缓存版本更新为 `kalimba-tone-trace-v1.0.1`。

## 1.0.0 - 2026-06-13

- 建立 `Kalimba ToneTrace / 卡林巴循音` 工程结构。
- 将原单文件网页拆分为 `index.html`、`src/app.js`、`src/songs.js`、`src/pitch.js` 和 `src/styles.css`。
- 新增 PWA 配置、离线缓存、手机桌面图标和旧地址跳转页。
- 新增本地自绘木质卡林巴图标。
- 保留卡林巴跟练、示范播放、麦克风识别、横屏练习和琴键宽度缩放等核心功能。
