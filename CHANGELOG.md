# 更新日志

本项目从 `1.0.0` 开始记录版本变更。每次发布前都要在这里新增对应版本的条目，并同步更新 README 中列出的版本号位置。

## 2.2.16 - 2026-07-21

- 新增《蜜雪冰城主题曲》主旋律版与和弦版，依据 1=D、2/4、115 BPM 双手简谱转写为 21 音 C 调卡林巴版本，保留弱起、附点与完整中文主题段。
- 新增《诀别书》主旋律版与和弦版，以完整 85 BPM 演奏脚本还原 334 个事件，并用卡林巴谱、双手简谱和 C 调指弹谱交叉复核主题、节奏与 `Fmaj7-G-Em7-Am7` 和声走向。
- 两首和弦版均加入稀疏自动伴奏；《蜜雪冰城主题曲》保留开头 1.5 拍静音窗口，《诀别书》按源脚本提取小节和弦，避免自动伴奏覆盖快速旋律。
- 将 Service Worker 离线缓存版本更新为 `kalimba-tone-trace-v2.2.16`。

## 2.2.15 - 2026-06-22

- 新增《青城山下白素贞》主旋律版，依据中国曲谱网 1=C、4/4 简谱整理并保留主要唱段。
- 新增《See You Again》主旋律版与和弦版，依据吉他世界 1=C 钢琴简谱/五线谱预览整理主题 hook，并加入稀疏自动伴奏。
- 新增 Beyond《海阔天空》主旋律版与和弦版，依据打谱啦 1=F 简谱转为 C 调，并按源谱和弦级数加入稀疏伴奏与长空档静音。
- 更新《星际穿越主题曲》：主旋律版开头改为高音区 ostinato，并新增和弦版补入低音/和声支撑，让开头和尾段更接近原曲氛围。
- 将 Service Worker 离线缓存版本更新为 `kalimba-tone-trace-v2.2.15`。

## 2.2.14 - 2026-06-18

- 新增儿歌《大雨小雨》主旋律版，依据虫虫乐谱 1=D、4/4 简谱转写为 21 音 C 调卡林巴可弹版本。
- 从内置曲库移除 `Hollow Knight Main Theme`、《女儿情》、《学猫叫》和《栀子花开》，包含对应和弦版文件。
- 将 Service Worker 离线缓存版本更新为 `kalimba-tone-trace-v2.2.14`。

## 2.2.13 - 2026-06-17

- 重新从吉他世界 C 调谱、CNBlogs 编码简谱和维基乐谱片段复核《送别》，修正旧版误读伴奏行/八度导致的跑调问题，并同步更新主旋律版、和弦版与句尾休止。
- 重新从中国曲谱网、吉他世界 C 调谱和搜狐双手简谱复核《童年》，修正旧版低音点与中音区混读导致的八度错位，并按 4/4 节奏重写第一段主旋律与和弦版。
- 将《天空之城》主旋律版和和弦版更新为中国曲谱网 `p343924` 的 17 音 C 调卡林巴演奏谱版本。
- 修正旧谱第 5、13、16、23、24 小节与该谱源不一致的问题，并移除旧混合来源中多出的结尾小节。
- 同步更新《天空之城》和弦版的事件长度、稀疏伴奏、节奏来源和 review 标记。
- 新增《云烟成雨》主旋律版，依据中国曲谱网 17 音 C 调卡林巴卷帘谱和清晰简谱交叉整理，伴奏/和弦暂未编码。
- 新增《菊次郎的夏天》主旋律版，依据中国曲谱网 17 音 C 调卡林巴单音简谱整理。
- 新增 `Hollow Knight Main Theme` 开头主旋律片段，按虫虫钢琴和 Scorely 预览谱源整理为 21 音 C 调卡林巴可弹旋律。
- 新增《星际穿越主题曲》主旋律版，依据 Doc88 三页简谱整理；后半段 4/4 与三连音分解和弦按当前曲库能力做可弹性简化。
- 拉取并核对 Quark 文档中的《好一朵美丽的茉莉花》拇指琴简谱；该谱与内置《茉莉花》开头旋律一致，但属于短版来源，因此仅作为交叉来源记录，不覆盖现有 30 小节版本。
- 将 Service Worker 离线缓存版本更新为 `kalimba-tone-trace-v2.2.13`。

## 2.2.12 - 2026-06-14

- 修复横屏练习顶部控制栏在移除麦克风与计分控件后仍沿用旧列宽，导致“示范播放”、速度条和“伴奏开”挤在一起的问题。
- 横屏控制栏改为显式列位：曲目、开始、示范、速度、伴奏分别占独立区域，按钮文字保持在按钮内居中。
- 将 Service Worker 离线缓存版本更新为 `kalimba-tone-trace-v2.2.12`。

## 2.2.11 - 2026-06-14

- 移除麦克风检测、音高识别、麦克风权限请求和相关诊断显示，练习页不再向用户申请录音权限。
- 练习模式改为纯跟谱和屏幕琴键演奏，不再统计命中、连击、miss 或按键判定结果。
- 屏幕琴键改为按下即触发声音，采样未完成预热时先用本地合成拨弦声即时响应，降低触摸演奏延迟。
- 示范播放期间屏幕琴键禁用，避免误触影响示范听感。
- 将 Service Worker 离线缓存版本更新为 `kalimba-tone-trace-v2.2.11`。

## 2.2.10 - 2026-06-14

- 手机横屏练习时会主动尝试进入浏览器全屏；如果浏览器要求用户手势，则在旋转后的下一次点击自动补发全屏请求，减少手动打开全屏的步骤。
- PWA manifest 改为优先使用 `fullscreen` 显示模式，安装到桌面后更接近沉浸式横屏练习。
- 新增独立“支持与共建”页面，曲库页提供入口，用于支付宝打赏、GitHub 共建引导和 AI 化曲谱流程介绍。
- 统一全站字体栈和字重层级，减少不同页面、按钮和标题之间的粗细跳变。
- 曲库页移除冗余“返回练习”按钮，并将“支持与共建”改为带咖啡图标和简短说明的低调入口。
- 支持与共建页新增开发故事，说明从音乐零基础、看不懂简谱到用 AI 做下落块练习工具和真实卡林巴采样的过程。
- 开发故事补充时间成本和 AI token 成本对比：目前开发用时约一天，token 消耗折算约五六百刀。
- 将 Service Worker 离线缓存版本更新为 `kalimba-tone-trace-v2.2.10`。

## 2.2.0 - 2026-06-13

- 新增 V2 内置曲目的 `rhythm.restWindows` 节奏间歇数据，用于记录主旋律 1 拍以上的呼吸和停顿。
- 自动伴奏生成改为 rest-aware，默认不在主旋律长空档内填充伴奏，改善句尾停顿和弱起间歇听感。
- 为《萱草花》《因为爱情》《安和桥》《大鱼》《天空之城》的节奏间歇写入外部复核来源。
- 新增 `npm run report:rhythm`，用于报告长空档、伴奏事件数量和空档内伴奏冲突。
- 校验器新增 V2 `rhythm` 结构检查，并禁止 `silent` 空档内出现伴奏事件。
- 将 Service Worker 离线缓存版本更新为 `kalimba-tone-trace-v2.2.0`。

## 2.1.2 - 2026-06-13

- 手机横屏练习界面保留“暂停”按钮，放在当前曲目和“换曲”按钮之间。
- 收紧下落音符块的音名与简谱数字间距，减少单个音符块的纵向占用。
- 横向节奏条音符块改为同时显示音名和简谱数字，并修复紧凑横屏下底部边框被裁切的问题。
- 曲库列表的“开始练习 / 继续练习”操作改为紧凑圆形播放图标，减少歌曲卡片横向占用。
- 修复跟练时目标列竖向阴影每帧闪动的问题，高亮只在目标音切换时移动。
- 在横向节奏条底部加入可拖动的整曲进度条，支持鼠标、触控和键盘定位。
- 进一步调整目标列竖向阴影切换时机，只在音块接近判定线时切换，避免长间隔或长音提前高亮造成闪动感。
- 更新 PWA 应用图标为黑白简笔画卡林巴，保留彩色键位并减少细碎零件。
- 将 Service Worker 离线缓存版本更新为 `kalimba-tone-trace-v2.1.2`。

## 1.1.0 - 2026-06-13

- 从用户录制的 21 音卡林巴录音中切分真实采样，按页面琴键从左到右的音名保存为 `d6.mp3` 到 `e6.mp3`。

## 1.0.5 - 2026-06-13

- 添加歌曲保存时新增短暂的 `AI thinking......` 加载弹窗，让导入 JSON 的反馈更明确。
- 导入期间会暂时禁用保存按钮和 JSON 输入框，避免重复点击。
- 添加本地曲库管理，可删除本机导入歌曲，系统内置歌曲不可删除。

## 1.0.4 - 2026-06-13

- 新增曲库难度分级，内置和本地导入歌曲都会带有 `easy`、`medium`、`hard` 难度。
- 曲库页新增“全部 / 简单 / 中等 / 困难”分组，搜索结果会先全局过滤，再按难度分组展示。
- 添加歌曲提示词和 JSON 导入支持 `difficulty` 字段，缺省时会根据速度、密度、跨度和跳音估算难度。
- 从曲库选择歌曲进入练习页时，手机竖屏下会直接启用软件横屏练习模式，省去二次点击横屏提示。
- 将 Service Worker 离线缓存版本更新为 `kalimba-tone-trace-v1.0.4`。

## 1.0.3 - 2026-06-13

- 新增本地歌曲导入能力，曲库页可复制 AI 提示词并粘贴标准 JSON 保存到本地曲库。
- 新增 `src/song-store.js`，统一管理内置曲库、本地曲库、JSON 校验和简谱 `notation` 到练习 `steps` 的编译。
- 导入 JSON 仅通过 `JSON.parse` 解析，并校验 C 调、音域、节拍和必填字段。
- 练习页和曲库页改为读取“内置歌曲 + 本地歌曲”的合并曲库。
- 将 Service Worker 离线缓存版本更新为 `kalimba-tone-trace-v1.0.3`。

## 1.0.2 - 2026-06-13

- 练习页移除原生歌曲下拉框，改为显示当前曲目并通过“换曲”进入曲库。
